Какие профессиональные компетенции формируются в ПМ.02?
Разработка модулей, интеграция, тестирование, работа с Git, проектирование БД, создание графического интерфейса.

2. Почему выбран стек: Python + MySQL + tkinter? Какие альтернативы?
Почему: Python простой, MySQL бесплатный и надёжный, tkinter встроен в Python.

Альтернативы: вместо Python — Java или C#, вместо MySQL — PostgreSQL или SQLite, вместо tkinter — PyQt или веб-интерфейс.

3. Что такое ER-диаграмма? Для чего она нужна?
Это схема, показывающая таблицы, их поля и связи между ними. Нужна для проектирования базы данных до её создания.

4. Какие типы связей между таблицами существуют? Примеры из своего варианта.
Три типа: один к одному (паспорт — человек), один ко многим (автор — книги), многие ко многим (студенты — курсы через таблицу связи).

5. Что такое Git? Чем отличается git commit от git push?
Git — система контроля версий. commit сохраняет изменения локально на компьютере, push отправляет их на удалённый сервер (GitHub).

6. Как проверить, что Python установлен и tkinter работает?
Ввести в консоли python --version, затем python -m tkinter. Если откроется тестовое окно — всё работает.

7. Какие меры безопасности нужно соблюдать?
Ставить антивирус, не открывать подозрительные ссылки, использовать сложные пароли, блокировать компьютер при уходе, делать резервные копии.

8. Что означает IP-адрес 127.0.0.1?
Это локальный адрес, который указывает на тот же компьютер. Используется для тестирования программ без реальной сети.

9. Как добавить файл в репозиторий Git (последовательность команд)?
git add имя_файла → git commit -m "сообщение" → git push

10. Зачем нужен файл README.md в проекте?
Чтобы описать проект: что он делает, как установить и запустить, как им пользоваться. На GitHub он показывается на главной странице репозитория.
